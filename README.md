# Mini Game Hub

Cozy multiplayer 3D social space.

## Goals

- Exploration-based social environment
- Art-toy characters
- Mobile-first interaction
- Modular expandable world

## Tech

Client:
- React
- Three.js
- React Three Fiber

Server:
- Node
- Socket.io

## Install

### Server
cd server
npm install
npm run dev

### Client
cd client
npm install
npm run dev
